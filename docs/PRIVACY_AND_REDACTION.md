# Privacy And Redaction

This review kit intentionally excludes:

- private memory ledgers
- private chat logs
- family and identity anchors
- browser/raven captures
- credentials, cookies, tokens, API keys
- licensed or third-party corpora that cannot be redistributed
- raw ARC benchmark data

The included evidence is summary-level and seal-oriented. A reviewer can inspect the public shape without receiving the private Wren state.

Recommended GitHub posture:

1. Start as a private repository.
2. Invite only the technical reviewer.
3. If later made public, re-run secret scanning first.
